Knowledge Block

 

I'm currently reading Mastering PHP 7 by Branko Ajzele, and so far it has been a pretty enjoyable ride. The author introduces many concepts that are bound in object orientation, so if you are new to that methodology of programming it might be a bit harder to get your gears rolling. However, if you are already comfortable with OO then you will find the material right at home, and will be able to pick up the language much more easily.
